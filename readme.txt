实验报告在当前文件夹中，文件名为“BY1906008_贺磊_计算几何实验报告.docx”
代码用python实现，存放在source文件夹中。
source文件夹中：
1. range_tree.py中包含实现程序的所有代码。直接在命令行中执行该脚本文件将加载70k.npy中的七万个点并构建区间树。之后会进行10000次随机生成的区间查询并统计区间查询与暴力循环求解所用时间的长短。具体代码实现见range_tree.py
2. 20k.npy, 40k.npy, 70k.npy三个文件存储随机生成的，坐标范围在[-1,1]的点，文件名表明该文件中包含的点的数量。也可以通过range_tree.generate_points函数生成随机分布的点作为实验数据。
